# screenshots/ — Isaac Sim native experiment

Visual evidence captured during each run.

**Naming convention:** `NN_taskID_NN_short_slug.png` (e.g. `01_spawn_cube_03_after_step.png`) so screenshots within one task sort chronologically.

Recommended capture moments:

- **Before** — initial scene state.
- **During** — at least one mid-run frame when the agent is performing the action.
- **After** — final scene state at the moment the task is judged complete (or failed).

Keep raw screenshots here. Curated / annotated versions used in the thesis go in `../../figures/`.
