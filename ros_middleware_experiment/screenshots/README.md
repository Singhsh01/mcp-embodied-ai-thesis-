# screenshots/ — ROS-middleware experiment

Visual evidence captured during each LIMO run.

**Naming convention:** `NN_taskID_NN_short_slug.png`.

For each task, capture at least:

- **Scene before** — robot pose + relevant scene objects.
- **`ros2 topic list`** — proves the ROS graph is present (a one-time capture per session is fine).
- **Mid-run frame** — the robot in motion.
- **Final state** — pose at the moment the task is judged complete.

Curated/annotated versions used in the thesis live in `../../figures/`.
