# thesis_pdf/

The compiled thesis document and any final-submission artefacts (signed cover sheet, abstract, declaration of authorship, etc.).

**Recommended contents:**

- `thesis.pdf` — the final compiled thesis.
- `abstract.md` — short standalone abstract (also embedded in the PDF, kept here for indexing).
- `bibliography.bib` *(optional)* — if you want the BibTeX file to live next to the PDF for archival.
- `submission/` — anything that was uploaded to the examination office (cover sheet, plagiarism declaration, etc.).

LaTeX intermediates (`.aux`, `.log`, `.toc`, `.bbl`, …) are gitignored at the workspace root.
